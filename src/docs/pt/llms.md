# Compatibilidade com IAs e LLMs

O Sawabona Forms permite que desenvolvedores gerem rapidamente formulários dinâmicos em React usando schemas JSON estritos. Por depender fortemente de schemas personalizados processados em tempo de execução, este framework é excepcionalmente adequado para a geração e integração de inteligência artificial.

Se você está utilizando Modelos de Linguagem Grande (LLMs, como ChatGPT, Claude, etc.) para construir ou gerenciar sua aplicação, pode alimentá-los com diretrizes específicas para garantir que eles criem objetos `FormSchema` 100% compatíveis.

## Arquivos de Instrução

Preparamos dois formatos de instrução detalhando nossos recursos da Referência da API e Regras de Ouro para as IAs. Eles são funcionalmente idênticos e servem para ensinar o LLM a como escrever esquemas precisos e complexos usando o `Sawabona Forms`.

Você pode baixar esses arquivos e enviá-los diretamente através dos prompts dos seus LLMs, nas instruções personalizadas do ChatGPT ou frameworks agenticos (Langchain, Cursor, etc).
