# Introduction

**Sawabona Forms** is a powerful, schema-based form builder for React. It is designed to create Typeform-like conversational experiences with minimal effort.

## Key Features

- **JSON Schema Driven**: Define your entire form logic in a single JSON object.
- **Conversational UI**: Smooth transitions and focused inputs.
- **Themable**: Fully customizable colors, fonts, and branding.
- **Responsive**: Works perfectly on mobile and desktop.
- **Validation**: Built-in validation for common use cases.
- **Accessible**: Semantic HTML and keyboard navigation.

## Philosophy

We believe that forms should be conversations, not interrogations. By focusing on one question at a time, we increase completion rates and improve data quality.
